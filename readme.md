# Web Scrapers

We scrape the website https://books.toscrape.com/ but you can change the code to scrape something else.

## Node.js & Puppeteer Usage

Install dependencies

```bash
cd web-scraper-node
npm install
```

Run the file

```bash
node scrape
```

## Python & BS4 Usage

Install dependencies

```bash
cd web-scraper-python
pip install -r requirements.txt
```

Run the file

```bash
python scrape.py
```
